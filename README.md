# FootballTeamsTestCaseForPiA


What's this about:

A Spring Boot project that you can store football players who belong to football teams.

Requirements:

- A service that lists football teams,

- Listing players of a team with the team ID,

- Deleting and updating a team with its ID,

- Adding players to a team and deleting them.
